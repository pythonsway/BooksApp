API

Query for book details and reviews programmatically via website’s API

API Access:

You can make a GET requests from adress /api/<isbn> route, where <isbn> is an ISBN number. It will return a JSON response containing the book’s title, author, publication date, ISBN number, review count, and average score.

In format like in example below:

{
"average_score": 5.0
"author": "Doug Lloyd"
"isbn": "1632168146"
"review_count": 28
"title": "Memory"
"year": 2015
}
